# Projeto P3 front em React

comando para installar o vite
```sh
  # clone no projeto
  npm install -g create-vite

```

comandos apos instalar o vite
```sh
  # clone no projeto
  git clone https://github.com/mateussf99/projeto_react
  #instalar os modulos do node
  npm i
  #rodar o localhost
  npm run dev

```
